Graph Traversal Breadth First Search Optimization

Breadth First Search is a significant implementation of graph processing. I tried deeply in the optimization of breadth first search through parallelization, blocking, and other algorithms, such as hybrid algorithm and Dijkstra’s algorithm.

Smoothie Dispenser

Smoothie Dispenser is a remote food drop system that consumers can use to drop smoothie onto smoothie container by pressing a remote button on their phone. There are several elements: servos, thermistor, accelerometer, WebCam, Lidar, alphanumeric display.

Cruise Control

We built a cruise that we can control through a website we made. We are able to control its direction. The system has been set to move in a range of speed through several techniques. For further explanation, please reach to project page.

Electronic Voting

E-voting will be enabled by unique devices (fobs) given to all registered voters. Voting will consist of creating a secure near field communications link between two fobs. Once the vote has been received, it should be communicated to the Poll Leader. It will send the results to a server which will log the vote to a database. Poll results will be accessible via the server via http, and there is a reset button that will restart the vote.

Hurricane Box

Hurricane Box is suitable for recording data experienced over an episodic weather event. The box records what’s happening by using accelerometer and thermistor and allows playback based on the time series of recordings with graphs to a personal domain by using WebCam.

Tactile Internet

We connected the 3 sensors(thermistor, ultrasonic sensor, IR range sensor) to the ESP32 using ADC, and output these sensor values and graph them on a website to build a tactile Internet.

Dog Cat Classifier

Dog and cat classifier is a basic machine learning that I implement probability through many training data to make the computer know how to figure out whether it is a dog or cat.

Multicycle CPU

Multicycle CPU is a pipelined CPU from computer organization. It includes several functions, such as ALU, rigisters, sign extension, etc.

GreenMate App

This app is to advocate eco-friendly activities. The main function includes a step counter, recycling database, and counter that users input in the number of recycling items.

Pokemon Game

This program is a game allows player to train their own pokemons and battle. There are stamina and money system while these calculate automatically after training or refilling the stamina.

Travel Recommendation App

The project imported a large database and gave the top 10 recommended countries to visit based on different proportion of air cleanness and popularity.

Truss Design

We used straws to build the truss design. We need to calculate the force and estimate the maximum weight the truss can hold in certain structure. We analyze with Matlab code. The data will be helpful in visualizing the truss that will be built and determining the exact dimensions of the truss.
